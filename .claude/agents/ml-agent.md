## Issue #12 — Training

**Data sources:**
- USFS Fire Occurrence Database (historical US fire data)
- CAL FIRE historical incidents
- Synthetic data generated by `scripts/generate_training_data.py`

**Features:** fire size, spread rate, wind speed, population density, nearest station distance, time of day, terrain type
**Label:** optimal resource allocation (stations dispatched, aerial support Y/N, mutual aid Y/N)
**Model:** XGBoost classifier with confidence output (use `predict_proba`)

**Training script entry point:**
```python
# ml/dispatch_model/train.py
if __name__ == '__main__':
    # SageMaker passes hyperparams via CLI args
    parser = argparse.ArgumentParser()
    parser.add_argument('--max-depth', type=int, default=6)
    ...
```

## Issue #13 — Deployment

After training completes:
```python
from sagemaker.xgboost import XGBoostModel
model = XGBoostModel(model_data='s3://...', role=role, framework_version='1.7-1')
predictor = model.deploy(instance_type='ml.m5.large', initial_instance_count=1)
```

Verify inference returns a confidence score between 0 and 1.

## Issue #14 — Bedrock advisory prompt

```python
ADVISORY_PROMPT = """
You are an emergency management AI generating a wildfire evacuation advisory.
You must be accurate, calm, and actionable. Never speculate beyond the data given.
```
